Exercício: anotação das tags aprendidas

Índice de Tags exploradas:

  1. h1 até h6
  2. p
  3. mark
  4. i, u, strong
  5. ul, ol, li
  6. a
  7. sup, sub, small
  8. blockquote

h1 até h6

As tags de heading são as que acompanham os títulos da página e de seções. São utilizadas principalmente para indicar o tema que o trecho seguinte falará a respeito. Os tamanhos variam conforme a numeração, onde h1 é a de maior prioridade e é um "mito" que não pode se utilizar dois h1s na mesma página, apenas não é recomendável.

p, span

A tag de parágrafo é utilizadas para textos mais extensos do que um título, podemos também encontrar a prática da junção do span dentro das tags de parágrafo, ou por vezes, até mesmo substituir a tag de parágrafo pela de span.

mark

A tag de marcação é utilizada para destacar um trecho como um marca-texto.

i, em / strong, b / del / u

As tags semânticas podem serem confundidas com as sem semântica alguma, portanto: strong é semântico e b não, em é semântico e i não. Já a tag u é para sublinhado e não semântica, porém não possui o seu lado semântico. A tag del servirá para traçar um texto antigo que fora excluído, como nas leis.

ul, ol e li

São as tags para trabalhar com uma listagem de itens, enquanto ul é para tratar de listas sem ordenação, a de ol é de maneira ordenada. Da mesma forma, as li são os itens individuais.

a

As tags de links são tratadas por a, que significa âncora. Sua síntaxe aceita não somente itens, mas como também aceita a navegação de seções pela página. Importante se atentar pelos demais atributos possíveis nesta tag, como a de target e title.

sup, sub e small

Enquanto sup e sub tratam do posicionamento da palavra na parte de cima ou de baixo, respectivamente. A tag small é uma tag que trata os caracteres em menor tamanho.

blockquote

Utilizadas para a exposição de citações, podendo ser frases de autores externos com os devidos créditos ou, em um exemplo prático, depoimentos de clientes para a empresa.
